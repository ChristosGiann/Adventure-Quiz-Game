# Adventure Quiz Game

A 3D Unity WebGL quiz game for kids, developed as a technical interview project.

## Play the Game

You can play the WebGL version here:

https://chrisgian.itch.io/quiz-game-for-kids

## About the Project

Adventure Quiz Game is a small 3D educational quiz game built in Unity.
The player explores a 3D environment, interacts with objects, and answers quiz questions through an in-game UI system.

The project was originally created as part of a Unity technical assignment for a company that develops quiz-based educational content for children.

## Features

* Main menu with Start and Exit options
* 3D environment with free third-person player movement
* Third Person Controller using Unity Starter Assets
* Object interaction system
* Dynamic UI messages
* Quiz system with multiple-choice questions
* Score tracking
* Health/mistake system based on wrong answers
* Victory and defeat screens
* Background music and interaction sounds
* Particle effects
* WebGL browser build hosted on itch.io

## Technologies Used

* Unity 2022.3
* C#
* WebGL
* TextMeshPro
* Starter Assets - Third Person Controller
* Cinemachine
* ScriptableObjects for quiz data

## Project Purpose

This project was created to demonstrate basic Unity gameplay programming, UI handling, scene management, player interaction, and a dynamic quiz system suitable for an educational game prototype.
